# fishboxen-dot-net

Homepage for Thomas Ian Fisher.

Built using Vite, React, and Jest.

Hosted on GitHub pages at [fishboxen.net](https://fishboxen.net/).

## Build the app

```bash
    npm build
```

## Run the app locally

```bash
  npm run dev
```

- Coverage reports stored in [/reports/coverage-report](./reports/coverage-report)
- Unit test reports stored in [/reports/unit-test](./reports/unit-test)

## Test the app

### Unit tests

```bash
    npm run test
```

### E2E tests

```bash
 npx playwright test
```
